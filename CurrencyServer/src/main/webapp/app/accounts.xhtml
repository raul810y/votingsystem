cuentas corrientes de usuario

Las etiquetas son un mecanismo para poder determinar en qué tipo de conceptos se puede gastar el dinero. Por ejemplo, el dinero con la etiqueta energía sólo puede emplearse en conceptos que tengan asociada esa etiqueta.

La etiqueta gastos libres es una etiqueta especial, el dinero asocidado a la misma se puede emplear en cualquier tipo de etiqueta

Cada usuario tiene asignado un codigo IBAN que tiene asociadas tantos depósitos como etiquetas reconocidas por el sistema.

El dinero de un depósito asociado a una etiqueta sólo se puede emplear en conceptos que tengan asociada esa etiqueta, exceptuando el dinero del depósito asociado a la etiqueta de gastos libres que se puede emplear en cualquier tipo de etiqueta.

Cuando un usuario RECIBE un ingreso CON etiqueta el importe se guarda en su depósito para esa etiqueta y sólo puede utilizarse en conceptos asociados a esa etiqueta

Cuando un usuario HACE un ingreso a otro usuario el importe se extrae del depósito de gastos libres

Cuando un usuario HACE un ingreso CON etiqueta el importe se extrae del depósito de esa etiqueta, si el importe excede el contenido del depósito lo que falta se extrae del depósito para gastos libres
